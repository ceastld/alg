## 数学符号说明

- $P(A)$ - 事件A的概率
- $P(A|B)$ - 在事件B发生的条件下事件A的概率
- $E[X]$ - 随机变量X的期望
- $\text{Var}(X)$ - 随机变量X的方差
- $\text{Cov}(X,Y)$ - 随机变量X和Y的协方差
- $\rho_{XY}$ - 随机变量X和Y的相关系数
