# Jacobi 迭代法求解线性方程组

## 题目
线性方程组：{ 4x + y = 6, x + 3y = 3 }
初始解 (x₀, y₀) = (0, 0)，进行一次 Jacobi 迭代后，(x₁, y₁) 是：

A. (0, 0)  
B. (2, 0.5)  
C. (1, 1)  
D. (1.5, 1)

## 解法

### 1. 矩阵形式
将线性方程组写成矩阵形式 Ax = b：

```
A = [4  1]    b = [6]    x = [x]
    [1  3]        [3]        [y]
```

### 2. Jacobi 迭代公式
对于方程组 Ax = b，Jacobi 迭代公式为：

```
x_i^(k+1) = (b_i - Σ(A_ij * x_j^(k))) / A_ii  (j ≠ i)
```

### 3. 计算过程
初始解：x₀ = 0, y₀ = 0

**第一次迭代：**

对于 x₁：
```
x₁ = (b₁ - A₁₂ * y₀) / A₁₁
   = (6 - 1 * 0) / 4
   = 6 / 4
   = 1.5
```

对于 y₁：
```
y₁ = (b₂ - A₂₁ * x₀) / A₂₂
   = (3 - 1 * 0) / 3
   = 3 / 3
   = 1.0
```

### 4. 结果
一次 Jacobi 迭代后的解为：(x₁, y₁) = (1.5, 1.0)

### 5. 验证
将结果代入原方程组：
- 4(1.5) + 1(1.0) = 6 + 1 = 7 ≠ 6
- 1(1.5) + 3(1.0) = 1.5 + 3 = 4.5 ≠ 3

注意：一次迭代通常不会得到精确解，需要多次迭代才能收敛。

## 答案
**D. (1.5, 1)**
